Questo sito e' hostato con il mio vecchio telefono

Procedura:

  • 1. *Installare F-Droid*
  • 2. *Scaricare termux*
  • 3. *pkg update && pkg upgrade*
  • 4. *pkg install apache2 openssh neofetch nmap git neovim wget curl*
  • 5. sshd; apachectl; per startare apache e ssh
    • (se serve una password usare comando passwd)
  • 6. *nmap -sV 127.0.0.1; whoami;* per info su login (porta user e ip)
  • 7. nano o nvim $PREFIX/share/apache2/default-site/htdocs/index.html per modificare il sito
    • (attiva wake lock su termux per non chiudere l' attivita)
